# Plan: Asistente de bienvenida y traducción al castellano

**Created:** 2026-03-14 09:42
**Status:** Completed

## Objective

Añadir un wizard (asistente) de bienvenida que explique cómo utilizar la aplicación y traducir toda la interfaz de usuario al castellano.

## Context

La aplicación Interview Coach es un entrenador de entrevistas basado en IA construido con Blazor Web. Toda la interfaz estaba en inglés. Se requería:
1. Traducir todos los textos de la UI al castellano.
2. Añadir un asistente paso a paso (wizard) que explique al usuario cómo usar la aplicación, visible únicamente en la primera visita.

## Proposed Changes

### Traducción al castellano

- **App.razor**: `lang="en"` → `lang="es"`
- **Chat.razor**: Textos del hero de bienvenida, tarjetas de sugerencia y sus mensajes predefinidos
- **ChatHeader.razor**: Botón "New chat" → "Nueva conversación"
- **ChatInput.razor**: Placeholder del textarea, textos de accesibilidad (aria-label, title), texto de aviso legal
- **ChatMessageItem.razor**: Etiqueta "Assistant" → "Asistente", textos de carga y búsqueda
- **MainLayout.razor**: Mensaje de error de Blazor y botón "Reload" → "Recargar"
- **Error.razor**: Página de error completa traducida

### Nuevo componente WelcomeWizard

- **WelcomeWizard.razor**: Componente Blazor de asistente con 4 pasos:
  1. ¡Bienvenido a Interview Coach!
  2. Adjunta tu currículum
  3. Inicia la conversación
  4. Consejos para el éxito
- **WelcomeWizard.razor.css**: Estilos CSS con overlay, animaciones y diseño responsive
- Integrado en **Chat.razor** justo después del header

## Implementation Steps

1. Traducir todos los archivos de UI al castellano
2. Crear componente `WelcomeWizard.razor` con 4 pasos explicativos
3. Crear `WelcomeWizard.razor.css` con estilos coherentes con el diseño existente
4. Integrar `<WelcomeWizard />` en `Chat.razor`
5. Verificar la compilación y los tests existentes

## Acceptance Criteria

- [x] Toda la interfaz de usuario está en castellano
- [x] El wizard aparece en la primera visita
- [x] El wizard tiene 4 pasos con navegación (Anterior / Siguiente / Omitir / ¡Empezar!)
- [x] El wizard se cierra al hacer clic en el backdrop o en el botón de cerrar
- [x] Una vez cerrado/completado, el wizard no vuelve a aparecer (localStorage)
- [x] El build compila sin errores ni advertencias
- [x] Los tests existentes siguen pasando (8/8)

## Notes

- El nombre de la aplicación "Interview Coach" se mantiene en inglés al ser un nombre de marca.
- El system prompt del agente se mantiene en inglés para preservar la calidad de las respuestas del modelo de IA.
- El wizard utiliza los tokens de diseño CSS existentes (`--color-accent`, `--color-bg`, etc.) para una integración visual consistente.
- La detección de primera visita se implementa mediante `localStorage` via JS interop de Blazor.
